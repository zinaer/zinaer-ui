# 介绍

[Zinaer UI](https://zinaer.github.io/zinaer-ui) 是 uni-app 跨端 Vue 组件库，视觉规范来源于 [Vant Weapp](https://youzan.github.io/vant-weapp)，助力开发者快速搭建多端应用。
