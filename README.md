## 介绍

## 使用组件

以按钮组件为例，只需要在 `script` 中引入按钮对应的自定义组件即可

```js
import zrButton from '@/components/button/Index'

export default {
  component: {
    zrButton
  }
}
```

接着就可以在 `template` 中直接使用组件

```html
<zr-button type="primary">按钮</zr-button>
```
